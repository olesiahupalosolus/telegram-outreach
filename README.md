# Telegram Outreach

Веб-інструмент для масової розсилки персоналізованих повідомлень у Telegram через ваші особисті акаунти. Контакти і тексти повідомлень підтягуються з Google Sheets.

## ⚠️ Важливо перед використанням

1. **Telegram ToS:** масова розсилка незнайомим людям порушує [Terms of Service Telegram](https://telegram.org/tos) і може призвести до бану акаунта.
2. **GDPR / закон про персональні дані:** холодний DM комерційного характеру до людей, які не давали згоди, регулюється законом. Перевірте дозволи у вашій юрисдикції.
3. **Безпека секретів:** ніколи не комітьте справжні `api_id`, `api_hash`, файли `.session` або `accounts.json` із даними. Все чутливе — лише локально.

## Стек

- Python 3 + Flask
- Telethon (Telegram MTProto клієнт)
- pandas (читання Google Sheets через CSV-експорт)
- Деплой: Railway (gunicorn)

## Локальний запуск

```bash
# 1. Встановити залежності
pip install -r requirements.txt

# 2. Створити .env з власним секретом
cp .env.example .env
python -c "import secrets; print('SECRET_KEY=' + secrets.token_hex(32))" >> .env

# 3. Запустити
python app.py
```

Відкрити http://localhost:8080

## Як користуватись

1. **Додати Telegram-акаунт** — натисніть "+ Додати акаунт" і введіть `api_id`, `api_hash` з [my.telegram.org](https://my.telegram.org), номер телефону, код підтвердження (і пароль 2FA, якщо ввімкнено).
2. **Підготувати Google Sheet** — таблиця з двома колонками: `@username` отримувача і текст повідомлення. Відкрийте доступ "Переглядач за посиланням".
3. **Запустити розсилку** — вставте посилання на таблицю, виберіть колонки, акаунт і затримку (рекомендовано ≥20 сек), натисніть "Запустити розсилку".

## Деплой на Railway

Налаштовано через `railway.json` + `Procfile`. У змінних оточення Railway встанови:
- `SECRET_KEY` — згенерований випадковий рядок (32+ байти)

⚠️ **Не деплой у продакшн без обмеження доступу до самого веб-інтерфейсу** — зараз будь-хто з URL може запустити розсилку.

## Структура

| Файл | Опис |
|---|---|
| `app.py` | Flask-додаток, логіка авторизації Telegram і розсилки |
| `templates/index.html` | Веб-інтерфейс |
| `accounts.json` | Локальне сховище акаунтів (gitignored) |
| `sessions/` | Telethon session-файли (gitignored) |
| `.env.example` | Шаблон змінних оточення |
| `requirements.txt` | Python-залежності |
| `Procfile`, `railway.json` | Конфіг деплою на Railway |
